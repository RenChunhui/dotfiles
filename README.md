Install
---

> 目前几乎所有 `dotfiles` 配置都是放在 `~` 目录下，当开发环境一多，$HOME 目录下有太多的隐藏文件，个人推荐放在 $HOME/.config 目录下

``` sh
$ git clone https://github.com/renchunhui/dotfiles ~/.config/dotfiles
$ cd $HOME/.config/dotfiles
$ chmod +x ./install.sh
$ ./install.sh
```
