## Install

``` sh
$ git clone https://github.com/renchunhui/dotfiles.git ~/.config/dotfiles
$ chmod +x ./install.sh
$ ./install.sh
```

## What's in it?

- alacritty & tmux configuration
- git configuration
- homebrew configuration(only macOS)
- neovim configuration
- pacman configuration(only Archlinux)
- zsh configuration
- ...
